FlashInterface Readme
ActionScript Virtual Machine Communication
http://www.flashextensions.com/products/flashinterface.php

v2.0.0

FlashInterface provides the means to communicate directly between the Flash 8 and 9 AVMs. FlashInterface will allow you to move forward, building Flash 9 player applications while still using existing SWF resources from previous Flash player versions.
